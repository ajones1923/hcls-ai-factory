# Code of Conduct

This project operates in the domain of healthcare, life sciences, and artificial intelligence. Contributors and participants are expected to engage in a professional, respectful, and responsible manner consistent with the seriousness of this context.

Participation in this project is subject to the following expectations:

---

## Professional Conduct

- Treat all participants with respect and professionalism, regardless of role, background, or experience.
- Engage in technical discussion constructively and in good faith.
- Avoid personal attacks, harassment, or behavior that could reasonably be considered disruptive or exclusionary.

---

## Scientific and Technical Integrity

- Represent system capabilities, limitations, and results accurately.
- Avoid overstating performance, readiness, or suitability for clinical use.
- Cite external sources appropriately and respect intellectual honesty.

---

## Responsible Use of Data

- Do not contribute real patient data, identifiable personal information, or proprietary datasets.
- Example data must be synthetic, de-identified, or explicitly permitted for public use.
- Contributors are responsible for ensuring that submitted materials comply with applicable data-use restrictions.

---

## Research-Only Context

- This project is intended for research, educational, and infrastructure reference purposes.
- It is not intended to be used directly for clinical diagnosis, treatment, or decision-making without appropriate validation and regulatory approval.
- Discussions and contributions should clearly respect this distinction.

---

## Enforcement

Maintainers reserve the right to moderate discussions, remove contributions, or restrict participation when necessary to maintain a constructive and responsible project environment.

---

## Reporting

If you witness or experience behavior that violates this Code of Conduct, please report it by opening an issue or contacting the maintainers directly.

---

## Scope

This Code of Conduct applies to all project spaces, including:
- GitHub repositories (issues, pull requests, discussions)
- Communication channels associated with the project
- Project-related events or presentations

---

## Attribution

This Code of Conduct is designed specifically for healthcare and life sciences AI projects, emphasizing the unique responsibilities that come with working in this domain.

---

*This Code of Conduct exists to ensure that collaboration advances knowledge without compromising trust, safety, or dignity.*
