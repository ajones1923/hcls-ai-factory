Quick Start Guide
  1. Python Setup: Install Python dependencies in virtual environment
  2. Start Milvus: Launch the vector database (required for RAG)
  3. Check Ollama: Verify Ollama LLM server is running with Llama 3.1 70B
  4. Ingest VCF: Parse and embed variants into Milvus
  5. Launch Chat: Open the Streamlit chat interface
Service URLs
  • Milvus: localhost:19530 (gRPC API)
  • Ollama: localhost:11434 (OpenAI-compatible API)
  • Streamlit Chat: http://localhost:8501
  • Portal: http://localhost:5001
System Requirements
  • Docker with NVIDIA Container Runtime
  • NVIDIA DGX Spark or similar ARM64 GPU system
  • Ollama with llama3.1:70b model loaded
  • Python 3.10+
